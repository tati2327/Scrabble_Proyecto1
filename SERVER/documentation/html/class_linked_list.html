◆ Delete() [1/2]

void LinkedList::Delete ( char  data)

<Borra por dato

Aqui se tienen que usar tanto el Tmp como el Curr ya que como se va a borrar un elemento en una lista que solo tiene next se borra el Curr y el Tmp que siempre va a estar uno atras del Curr se le asigna como siguiente el siguiente del Curr por lo cual este queda "flotando y luego se borra"

Aqui se nota como el Tmp esta antes del Curr y se recorre la lista hasta encontrar el valor

Como se puede borrar por dato puede ser que no se encuentre, si el Curr llegara a ser nullptr despues de recorrer toda la lista significa que no lo logro encontrar

Se borrar el puntero deleted

Se asigna el puntero deleted el valor de Curr que es el puntero o nodo que se desea borrar, y se asigna a otra variable porque todvia necesitamos el puntero Curr y su siguiente para unir las listas despues de eliminar el elemento

Si se borra el Head se vuelve a asignar

Se hace delete que ya esta sobrecargado
